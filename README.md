Gboard

基于 V2board 与 Xboard 二次开发的高性能 Go 面板。
采用全新架构设计，支持多协议扩展与高并发场景，专注于大规模部署与稳定运行。

高性能 · 易部署 · 可扩展 · 面向大规模用户优化

🚀 项目特点

使用 Go 重构核心架构

支持多种主流协议

简化部署流程

支持多出口与自动负载均衡

面向高并发与大数据量场景优化

相比传统 PHP 面板架构，在性能与实时同步能力方面实现显著提升。

⚡ 一键部署

部署流程简单直接：

上传 gboard 可执行文件至服务器目录

赋予执行权限：

chmod +x gboard


运行安装：

./gboard


程序将自动完成：

MySQL 安装与初始化

Redis 安装与配置

面板环境部署

配置域名反向代理即可完成上线
