## 📘 Gboard 部署指南

### 💡 核心提示 | Tips

* **数据平滑迁移**：Gboard 完美兼容 **Xboard**、**V2board** 及 **SSPanel-UIM**。您可以通过 Web 页面或 CLI 工具实现一键数据转换。
* **服务守护**：在常规部署模式下，建议配合 `一键脚本` 使用，以确保进程在崩溃或重启后能自动拉起。
* **SSL证书**：建议在反向代理步骤中配置 SSL 证书（如 Let's Encrypt），以保证数据传输安全。

### 🚀 快速安装 | Installation

Gboard 将复杂的部署流程极简化。以下是以 **Debian 12** 为例的安装指南（请务必在防火墙或安全组中 **放行 8888 端口**）。

### 🛠️ 方法一：常规 Linux 部署 (CLI)

1. **上传文件**
将编译好的 `gboard` 二进制文件上传至服务器目录（建议路径：`/root/gboard`）。
2. **赋予执行权限**
```bash
chmod +x ./gboard
```
3. **执行安装**
```bash
./gboard

```
* **交互式安装**：进入 CLI 模式后，根据提示选择适合您的安装方式。
* **特性支持**：支持远程数据库接入、本地环境一键安装。
* **备份恢复**：系统内置备份机制，支持在网站后台进行一键备份与数据恢复。
4. **配置反代**
安装完成后，使用 Nginx 或 Caddy 将您的域名反向代理至 `8888` 端口，即可完成部署。
---

### 🎮 方法二：宝塔面板部署 (BT Panel)

若您使用宝塔面板，请按照以下步骤操作（请确保宝塔安全组已放行 **8888** 端口）：

1. **新建项目**：在宝塔面板的【网站】菜单中，点击 **【Go项目】**。
2. **上传程序**：将 `gboard` 二进制文件上传至您创建的项目目录中。
3. **项目配置**：
* **项目执行文件**：选择您上传的 `gboard` 文件。
* **项目运行目录**：选择 `gboard` 所在的文件夹。
* **项目端口**：指定监听端口为 `8888`。
4. **开机启动**：建议设置开机启动，然后启动即可。
5. **环境变量 (重要)**：
* **初次安装**：无需填写“从文件加载环境变量”。
* **设置好启动后**：访问 `http://您的服务器IP:8888`，按照 Web 引导完成数据库和管理员配置。
* **安装完成后**：将宝塔项目配置中的 **“环境变量文件路径”** 指向生成的 `./env` 文件，确保重启后配置生效。
